Congratulations! you found the credits! This program was made in 2006 by Joachim Breitner and was updated and overhauled into the modern era by Wade D (ohio@GMX.us) in 2022. Screen Message is meant to be a tool you can use to message someone across a room without having to make noise, and a tool for making slides on-the-fly. I hid some extra styles in the code of the website. you can modify the url to use them (mirror mirror_red retro grainy black paper headline) I made it because I needed it, and tbh it took way too long to make, I suck at coding. if you make something cool with this tool or use it in some cool way, please email me. anyways, enjoy the tool. -Wade
